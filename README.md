# Simple CRUD with CI4 & Mysql

## Installation

Clone Project

```bash
  git clone https://github.com/attmhd/test_ci.git
```

Move to Project

```bash
  git checkout main
```

Rename env copy to .env

Install all libraries needed

```bash
  composer install
```

Run Project

```bash
  php spark serve
```
